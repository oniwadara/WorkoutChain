# WorkoutChain

A decentralized fitness routine and exercise program exchange platform for sharing workout knowledge on Stacks blockchain.

## Features

- Workout creation with comprehensive exercise routines
- Trainer intensity management and validation
- Fitness type classification system
- Duration specification and tracking
- Workout availability management and archival controls

## Smart Contract Functions

### Public Functions
- `create-workout` - Create new fitness workout with routine
- `archive-workout` - Archive workout routine (trainer only)

### Read-Only Functions
- `get-workout` - Get fitness workout details
- `get-trainer` - Get workout trainer information
- `get-total-workouts` - Get total created workouts
- `get-availability-status` - Get workout availability status

## Fitness Types
- Strength, Cardio, Flexibility, HIIT, Yoga, Pilates

## Intensity Levels
- Light, Moderate, Vigorous, High, Extreme

## Usage

Deploy the contract to create a fitness platform where trainers can share their workout routines and manage availability throughout the fitness community.
